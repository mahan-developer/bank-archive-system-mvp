# Bank Archive System (MVP)

A simplified enterprise-style document archive system inspired by banking workflows.

This project is built step-by-step to simulate a real-world internal system used for managing customers, cases, and documents with OCR indexing and workflow control.

## Scope (MVP)
- Customer and case management
- Document upload and archive
- OCR text indexing
- BPMN-based workflow design
- MVVM desktop architecture (planned)
- SQL Server database
